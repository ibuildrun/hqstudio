# ADR-003: Next.js для Web

**Дата**: 2025-01-20
**Статус**: Accepted

## Контекст

Необходимо было выбрать фреймворк для веб-приложения. Рассматривались:

1. **Next.js** — React с SSR/SSG
2. **Nuxt.js** — Vue с SSR/SSG
3. **SvelteKit** — Svelte с SSR/SSG
4. **Astro** — Multi-framework с SSG

Требования:
- SEO-оптимизация (маркетинговый сайт)
- Интерактивная админка
- Быстрая загрузка
- Деплой на GitHub Pages

## Решение

Выбран **Next.js 14** с App Router.

Стек:
- Next.js 14 — фреймворк
- React 18 — UI библиотека
- TypeScript — типизация
- Tailwind CSS — стилизация
- Framer Motion — анимации

## Последствия

### Положительные

- **SEO**: Server Components для статического контента
- **Производительность**: автоматическая оптимизация
- **DX**: отличный developer experience
- **Экосистема**: огромное количество библиотек
- **Static Export**: деплой на GitHub Pages
- **TypeScript**: строгая типизация

### Отрицательные

- **Размер бандла**: React добавляет вес
- **Сложность**: App Router имеет кривую обучения
- **Vercel-центричность**: некоторые фичи лучше работают на Vercel

### Нейтральные

- Требуется Node.js для разработки
- Частые обновления фреймворка
